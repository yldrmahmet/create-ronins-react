# Ronins React Nedir?

**Ronins React**, modern React projeleri oluşturan bir CLI aracıdır. Bir komut çalıştır, hemen kod yazmaya başla.

## Neden?

React kurulumu zaman alır. Vite, TypeScript, Tailwind, ESLint yapılandırman gerekir... Çok fazla iş.

**Ronins React** bunu çözer. Her şey önceden yapılandırılmış. En iyi uygulamalar dahil.

## Dahil Olanlar

| Araç | Ne Yapar |
|------|----------|
| Vite | Hızlı geliştirme sunucusu |
| React 19 | UI kütüphanesi |
| TypeScript | Tip güvenliği |
| Tailwind CSS 4 | Stillendirme |
| shadcn/ui | UI bileşenleri |
| React Router 7 | Yönlendirme |
| TanStack Query | Veri çekme |
| Zod | Doğrulama |
| ESLint + Prettier | Kod kalitesi |
| Vitest | Testler |
| Husky | Git hooks |

## Nasıl Çalışır

```bash
npm create ronins-react@latest uygulama-adım
cd uygulama-adım
npm install
npm run dev
```

Hepsi bu. 30 saniyeden kısa sürer.
